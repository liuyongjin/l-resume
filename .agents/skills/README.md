# Project Agent Skills

These skills are project-scoped copies of the globally installed skills used by l-resume agents (Cursor / skills CLI).

| Skill | Use when |
|-------|----------|
| `visual-testing` | Nuxt visual regression, screenshot diffs, Playwright baselines |
| `mobile-visual-qa` | Expo / Flutter style QA vs Nuxt brand tokens |
| `expo-native-ui` | Expo / React Native UI polish (safe area, shadows, HIG) |
| `screenshot-to-html` | Clone a screenshot into high-fidelity HTML |
| `screenshot-to-code` | Convert screenshots into React/Vue/HTML |
| `image-to-html` | Pixel-accurate vanilla HTML/CSS from an image |

Canonical path: `.agents/skills/` (skills CLI + Cursor project skills).

Project QA defaults: [../qa-project-context.md](../qa-project-context.md).
